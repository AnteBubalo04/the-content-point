1) Otvori folder product-site u VS Code.
2) Provjeri da su videi u product-site/assets/ i da se zovu:
   - video1.mp4
   - video2.mp4
   - video3.mp4
3) Pokreni index.html preko Live Servera.
4) Ako želiš finije poravnanje, otvori script.js i promijeni MATCH konstante:
   - v1 = pozicija animacije unutar Video1
   - v3 = pozicija animacije unutar Video3

Fajlovi:
- index.html -> struktura stranice
- style.css -> izgled i layout
- script.js -> scroll animacija, zoom, fade i scrub videa
- assets/ -> video datoteke
